# layout-ratio-profile

## Goal

Use proportion to improve clarity without forcing rigid math.

## When to use

Use after layout-region-mapping and before visual polish.

## Process

1. Select screen type.
2. Choose the matching ratio profile.
3. Identify primary and supporting regions.
4. Suggest ratio-safe adjustments.
5. Convert raw ratios to design-system tokens.
6. Reject changes that harm usability or accessibility.
