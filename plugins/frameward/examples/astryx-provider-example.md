# Example: Astryx Provider

User request:

> Use Astryx if it helps.

Frameward behavior:

1. Detect whether Astryx is installed.
2. If installed, read Astryx agent docs or CLI docs.
3. Use frame-first layout and Astryx components.
4. Use tokens instead of hardcoded values.
5. Explain result in plain language.

Plain-language result:

- I used the project’s UI library instead of inventing new styles.
- I made the screen structure clearer before changing individual elements.
- I kept spacing and colors consistent with the existing system.
