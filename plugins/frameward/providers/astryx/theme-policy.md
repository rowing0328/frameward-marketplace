# Astryx Theme Policy

Frameward should not ask users about tokens or theme internals.

Ask plain-language questions:

```text
Which direction feels closer?
A. Calm and minimal
B. Warm and soft
C. Dark and focused
D. Bright and expressive
```

Then map internally to available theme directions.

## Default

Use neutral as the default theme when bootstrapping, unless the user chooses another direction.

## Caution

Changing a theme can affect the whole product. Treat it as a higher-risk change than a single-screen polish pass.
