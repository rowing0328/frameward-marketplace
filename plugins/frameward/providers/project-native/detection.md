# Project-Native Detection

Look for:

- `components/`, `ui/`, `design-system/`, `packages/ui/`
- `theme.ts`, `tokens.ts`, `tailwind.config.*`, CSS variable files
- Storybook files
- Existing Button, Card, Input, Modal, Table components
- shared layout primitives
- package dependencies such as `@mui/*`, `@chakra-ui/*`, `@radix-ui/*`, `tailwindcss`, `class-variance-authority`

Record findings in `.frameward/design-system-provider.json` when possible.
