# Layout Ratio Model

Frameward uses proportion as a guide, not a rule.

Golden ratio and Fibonacci rhythm can help create balance, but they must not override usability, readability, accessibility, or existing design-system tokens.

## Useful ratios

- 1.618 : 1
- 61.8% : 38.2%
- 1 : 0.618
- Fibonacci-like spacing steps

## Where ratios apply

- Main content vs supporting content
- Form area vs summary/help area
- List area vs detail panel
- Visual area vs explanation area
- Card media vs text
- Mobile above-fold priority

## Token adjustment

Raw values should be converted to values that match the project’s token scale.
