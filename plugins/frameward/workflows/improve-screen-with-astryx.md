# Workflow: Improve Screen with Astryx

Use this only when Astryx is already installed or explicitly approved.

1. Run provider detection.
2. Confirm Astryx mode.
3. Read local Astryx agent docs or CLI docs. Use MCP docs only after explicit approval.
4. Map screen regions using frame-first layout.
5. Select Astryx components.
6. Use token-safe styling.
7. Avoid component prop guesses.
8. Run Frameward quality checks.
9. Explain result without Astryx internals unless helpful.
