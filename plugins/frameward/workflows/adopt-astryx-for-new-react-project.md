# Workflow: Adopt Astryx for New React Project

This workflow requires explicit user approval.

1. Confirm React project.
2. Confirm no mature existing design system.
3. Explain in plain language why a design system could help.
4. Ask for approval to install dependencies.
5. After approval, install Astryx packages.
6. After approval, run `npx astryx init --features agents --agent codex`.
7. Read generated docs.
8. Build a small representative screen.
9. Stop before broad migration unless approved.
