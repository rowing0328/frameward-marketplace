# Workflow: Improve Existing Screen

1. Run `client-intent-discovery`.
2. Run `design-system-provider-detection`.
3. Run `layout-region-mapping`.
4. Run `design-system-audit`.
5. Run `layout-ratio-profile` if balance or spacing is part of the issue.
6. Run `screen-improvement-plan`.
7. Implement small safe changes.
8. Run `responsive-check`, `accessibility-check`, and `state-coverage-check`.
9. Run `handoff-report`.

Default user-facing interpretation:

> I’ll keep the existing structure and style rules, then make the screen easier to read and easier to act on.
