# Landing Page Ratio Profile

## Goal

Persuasion flow

## Typical regions

hero message, product visual, proof, and action area

## Ratio guidance

Hero copy and visual split; proof and action sections should support the main promise.

## Useful ratio patterns

- 61.8% primary / 38.2% supporting
- 1.618 : 1 main/supporting split
- Fibonacci-like spacing rhythm
- Token-adjusted approximations

## Do not

- Force exact ratios when the project token scale disagrees.
- Make the screen less readable for mathematical neatness.
- Hide important controls below the fold just to preserve a ratio.
- Use ratio as decoration.
